# RiskLoom
A python tool kit that allows for stock analysis, risk management, and financial simulations

# Features

- Stock Price analysis and visualization
- Portfolio risk management tools
- Monte Carlo simulations for portfolio forecasting
- Bond pricing and yield calculations

# Installation

pip install RiskLoom

# Usage

from RiskLoom import portfolio

portfolio.simulate(portfolio_data, num_simulations=1000)
